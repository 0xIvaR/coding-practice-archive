# Playground Workflow

This directory is for your HTML practice.

## Folders

*   `assignments`: This folder contains the assignments I give you.
*   `my-work`: This is where you should work on your solutions. You can copy the assignment files from the `assignments` folder and modify them here.
*   `answers`: This folder will contain the solutions to the assignments. I will generate the answer files in this folder only when you ask me to.

## Workflow

1.  I will create a new assignment file in the `assignments` folder.
2.  You copy the assignment file to the `my-work` folder.
3.  You complete the assignment in the `my-work` folder.
4.  When you are ready, you can ask me to provide the solution, which I will place in the `answers` folder.
