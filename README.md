# Coding OS

Personal coding practice workspace for learning programming languages, data structures, algorithms, object-oriented programming, and LeetCode-style problem solving.

## Repository Structure

- `Languages/` - language-specific practice code and course notes.
- `Learning/` - topic-based learning folders, including DSA and OOP.
- `LeetCode/` - solved coding problems grouped by problem name.

## Current Focus

- Java fundamentals and OOP
- C++ problem solving
- DSA practice
- LeetCode solutions
- Python and web basics

## Notes

This repository is meant to track learning progress over time. Build outputs, editor settings, compiled binaries, and temporary files are ignored so the Git history stays focused on source code and practice notes.
